# Technology Stack

Atlas Systems is technology-agnostic in client engagements — we recommend the right tool for the problem — but the internal platform and most active client projects converge on the stack below.

## Cloud Platforms
- **Primary**: Kubernetes (managed and self-hosted)
- **Public clouds**: AWS, Microsoft Azure, Google Cloud Platform — we are certified partners with all three.
- **Greek hosting**: We also work with Greek datacentres for regulated workloads (banking, public sector).

## Programming Languages
- **Python**: backend services, data engineering, ML platforms
- **TypeScript / JavaScript**: frontend and Node.js backend
- **Java**: enterprise backend (Spring Boot)
- **Go**: high-performance services and DevOps tooling
- **SQL**: across PostgreSQL, MySQL, BigQuery, Snowflake

## Databases & Storage
- PostgreSQL, MySQL, MongoDB, Redis
- ChromaDB, Pinecone, Weaviate (for vector / RAG workloads)
- Snowflake, BigQuery (data warehousing)
- S3-compatible object storage

## DevOps Tooling
- **CI/CD**: GitLab CI, Jenkins, GitHub Actions
- **IaC**: Terraform, Ansible, Pulumi
- **Observability**: Prometheus, Grafana, Loki, OpenTelemetry
- **Secret management**: HashiCorp Vault

## Data & AI
- **Frameworks**: PyTorch, TensorFlow, scikit-learn
- **Orchestration**: Apache Airflow, Prefect
- **MLOps**: MLflow, Weights & Biases
- **LLM tooling**: LangChain, Ollama, vLLM, Hugging Face

## Collaboration
- **Source control**: GitLab (self-hosted) + GitHub Enterprise
- **Project management**: Jira, Confluence
- **Communication**: Slack, Microsoft Teams
- **Documentation**: Confluence, internal wiki
