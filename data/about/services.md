# Services

Atlas Systems offers four practice areas. Engagements are typically structured as either fixed-scope projects or long-running managed-services contracts.

## 1. Software Engineering
Custom software development for enterprise clients.
- Backend systems (Python/FastAPI, Java/Spring, Go)
- Frontend applications (React, TypeScript)
- API design and integration platforms
- Modernisation of legacy systems

## 2. DevOps & Site Reliability Engineering
Cloud infrastructure design, build, and run.
- Kubernetes platform engineering (clusters, GitOps, security)
- CI/CD pipeline design (Jenkins, GitLab CI, GitHub Actions)
- Cloud migration (AWS, Azure, GCP)
- Observability stacks (Prometheus, Grafana, OpenTelemetry)
- 24/7 SRE on-call coverage for managed-services clients

## 3. Data & AI
Data engineering and machine learning platform consulting.
- Data warehouse and lakehouse implementation (BigQuery, Snowflake, Spark)
- ML model lifecycle platforms (MLflow, Airflow)
- LLM and RAG application development
- Real-time streaming and event-driven analytics

## 4. IT Managed Services
Internal IT and corporate support — both for Atlas Systems itself and for selected smaller clients.
- Endpoint management
- Identity and access (Active Directory, SSO)
- Helpdesk and ticketing
- Security operations

## Typical Engagement Sizes
- **Small**: €50,000 – €150,000 (short-running fixed-scope projects).
- **Medium**: €150,000 – €500,000 (3–9 month delivery).
- **Large**: €500,000 – €1.5 million (multi-quarter delivery or annual managed-services contracts).
